# TaskFlow Backend

REST API for TaskFlow built with Node.js, Express, and PostgreSQL.

## Setup
1. Clone the repo
2. Run `npm install`
3. Create `.env` file with required variables
4. Run `node server.js`

## Environment Variables
- DB_USER
- DB_HOST
- DB_NAME
- DB_PASSWORD
- DB_PORT
- JWT_SECRET
- FRONTEND_URL
- BACKEND_URL
- BACKEND_PORT
